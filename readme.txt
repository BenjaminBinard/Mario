Vous incarnez Tails, le personnage de sonic. Le but est de trouver l'anneau pour terminer le jeu. 
Celui-ci se trouve en haut de la map. Le but est donc de monter. 
Vous ne pouvez pas traverser les plateformes par en dessous mais les bords droite et gauche sont collants. 
Cela permet de se hisser sur les plaeformes trop hautes.

Les plateformes excepté la première et les pics sont générés de manières aléatoire pour que le jeu soit quasiment infini.
